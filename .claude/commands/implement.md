# Implementar Plan

Este comando toma un plan ya escrito (por ejemplo, archivo `specs/*.md`) y lo implementa exactamente como está escrito, sin inventar alcance nuevo.

## Instrucciones

- Leer el plan completo
- Implementar cada paso en el orden indicado
- Si algo del plan no es posible tal como está, documentarlo como nota y proponer el ajuste mínimo
- Mantener los cambios limitados a lo que pide el plan
- **IMPORTANTE**: Si el plan incluye una sección "Comandos de Validación", ejecutar cada comando y reportar resultados (éxito/error)
